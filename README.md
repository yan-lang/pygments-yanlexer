# pygments-yanlexer

Yan lexer for [Pygments](http://pygments.org/).

## Install

    $ git clone https://github.com/yan-lang/pygments-yanlexer.git
    $ cd pygments-yanlexer
    $ (sudo) python setup.py install

### Verify

Verify that the package installed correctly by looking for the lexer "yanlexer" in the output of

    $ pygmentize -L lexers
